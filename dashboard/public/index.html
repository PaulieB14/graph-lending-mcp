Glossary

TVL (Total Value Locked)

The total USD value of assets deposited into a protocol's smart contracts. The primary measure of a protocol's size.

Total Deposits

USD value of all assets supplied by lenders. Lenders earn interest on deposits from borrowers.

Total Borrows

USD value of all outstanding loans. Borrowers pay interest and must maintain collateral above liquidation thresholds.

Supply APR

Annual percentage rate earned by lenders (depositors). Shown as the LENDER-side variable rate from the subgraph.

Borrow APR

Annual percentage rate paid by borrowers. Shown as the BORROWER-side variable rate. Higher = more expensive to borrow.

Liquidation

When a borrower's collateral value falls below the required threshold, anyone can repay part of the debt and claim collateral at a discount.

Market

A single lending pool for one asset (e.g., "Aave V3 USDC on Ethereum"). Each market has its own rates, TVL, and risk parameters.

Daily Active Users

Unique wallet addresses that interacted with the protocol (deposit, borrow, repay, or liquidate) in a given day.

Protocols Covered

Data Source

The Graph Protocol

All data comes from decentralized subgraphs indexed on The Graph Network. Queries go through the Gateway API to indexers staking GRT tokens.

Messari Standard Schema

Protocols use Messari's standardized lending schema (versions 1.3.0 through 3.1.0). This dashboard uses only universal fields that exist across all versions.

Update Frequency

Subgraph data updates with each new block on each chain. Dashboard queries are live — every page load fetches the latest on-chain state.
